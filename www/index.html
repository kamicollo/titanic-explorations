Titanic passenger survival

This web app allows you to try out explore relationships between variables in a real set of data taken from Titanic passenger survival competition on Kaggle.

Choose which variables to plot and how to plot them.

And when you feel like you explored enough - see how your imaginary passenger may have fared during the disaster.



Made by Aurimas using R and Shiny. Browse the source code on Github.
