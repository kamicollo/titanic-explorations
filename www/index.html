Predict Titanic passenger survival

This web app allows you to try out several ML algorithms on a real set of data taken from Kaggle.

You can choose the algorithms themselves, tinker with their tuning parameters as well as choose which features to incorporate. And, of course, see the results instantly!



Made by Aurimas using R, Shiny, and Plotly. Browse the source code at Github.
